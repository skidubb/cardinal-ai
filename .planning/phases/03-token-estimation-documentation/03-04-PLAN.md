Create `CE - Agent Builder/docs/` directory if it doesn't exist. Create `CE - Agent Builder/docs/BYPASS_PERMISSIONS.md` with the following sections:
- **Overview:** What `permission_mode="bypassPermissions"` does in `ClaudeAgentOptions`
- **Location:** `src/csuite/agents/sdk_agent.py` line 264
- **Rationale:** SDK agents run autonomously as part of multi-agent protocols via `asyncio.gather` — blocking on user approval per tool call would deadlock parallel execution across 48 protocols
- **MCP Servers Accessed:** Pinecone (read/write), Notion (read/write), SEC EDGAR (read-only), Pricing Calculator (read-only), GitHub Intel (read-only)
- **Risk Assessment:** Write access to Pinecone and Notion without human approval
- **Mitigations:** Role-based MCP mapping (`mcp_config.py`), `max_turns=15` limit, read-only external APIs, custom MCP servers with constrained operation sets
- **Review Cadence:** Note that MCP server additions should be reviewed for permission implications
